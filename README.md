# 📚 BibliotecaMunicipal

Sistema de gestión de biblioteca desarrollado en **Java** como proyecto de aprendizaje para aplicar los principios de la programación orientada a objetos (POO).

## 📖 Descripción

BibliotecaMunicipal es una aplicación que permite administrar los elementos básicos de una biblioteca, facilitando la gestión de usuarios, libros, préstamos y multas.

Este proyecto fue desarrollado con el objetivo de fortalecer conocimientos en Java, diseño de clases, encapsulamiento, herencia y organización del código siguiendo una arquitectura por capas.

## ✨ Funcionalidades

* 📚 Registro y gestión de libros.
* 👤 Registro y administración de usuarios.
* ✍️ Gestión de autores.
* 🔄 Administración de préstamos.
* 💰 Sistema de multas por préstamos.
* 🏛️ Administración general de la biblioteca.
* 📦 Organización del proyecto utilizando los paquetes:

  * `modelo`
  * `controlador`
  * `vista`

## 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio.

```bash
git clone https://github.com/JainerCampo/BibliotecaMunicipal.git
```

2. Abrir el proyecto en Visual Studio Code.

3. Compilar el proyecto.

4. Ejecutar la clase principal ubicada en:

```text
src/controlador/App.java
```
